# Location_Reminder

## Skills used:
```
1) Sending Notifications.
2) Firebase Authentication.
3) Using Google Maps and Styling them.
4) Using Geofencing.
5) MVVM Architecture.
6) Recycler View.
7) Room database.
8) Android testing using JUnit test, FakeDataSource, Espresso, Mocks and Koin.
9) Single activity architecture.
10) Data binding.
```
